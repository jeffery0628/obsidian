# JShell

JShell脚本工具是JDK9的新特性，当我们编写的代码非常少的时候，而又不愿意编写类，main方法，也不愿意去编译和运 行，这个时候可以使用JShell工具。在JShell中，可以编写Java代码，无需写类和方法，直接写方法中的代码即可，同时无需编译和运行，直接回车即可。

> JShell工具，只适合片段代码的测试
