# 字节

字节是常见的计算机中最小存储单元。计算机存储任何的数据，都是以字节的形式存储。
8个bit（二进制位） 0000-0000表示为1个字节，写成1 byte或者1 B。

-   8 bit = 1 B
-   1024 B =1 KB
-   1024 KB =1 MB
-   1024 MB =1 GB
-   1024 GB = 1 TB
